# UniWamp v0.1.0-alpha Release Notes

Status: alpha target

UniWamp `v0.1.0-alpha` is the first planned alpha release for early Windows validation. It is intended for local development testing and feedback, not production use.

## Alpha Scope

- Portable Delphi VCL dashboard for managing a local Apache, MariaDB, and PHP stack.
- Runtime-aware generation of Apache, PHP, MariaDB, vHost, and environment config files.
- Local virtual host management with browser, folder, editor, terminal, delete, and SSL actions.
- PHP runtime selection and PHP extension management.
- Script catalog UI for installing common PHP web applications.
- Terminal-style installation output for script runs.
- Developer tool launchers for local workflow support.

## Required Runtime Files

Unless using a package profile that bundles runtimes, provide these files in the UniWamp tree:

- `runtime/apache/bin/httpd.exe`
- `runtime/mariadb/bin/mariadbd.exe`
- `runtime/mariadb/bin/mysqladmin.exe`
- `runtime/php/<version>/php.exe`
- `runtime/php/<version>/php8apache2_4.dll` or a compatible Apache PHP module

Optional runtime/tool files:

- `runtime/apache/bin/openssl.exe`
- `runtime/nodejs/<version>/...`
- `bin/cmder/Cmder.exe`
- `home/adminer/index.php`

## Known Limitations

- This is an alpha validation target. Expect incomplete test coverage and rough edges.
- Clean-machine package validation must be completed before publishing the tag externally.
- Some script catalog entries depend on upstream installers and may require per-application fixes.
- Runtime folders are intentionally not tracked in Git.
- Generated website project folders under `www/` are intentionally ignored.
- The app is designed for local development workflows only.
